# 🧠 ESP Quiz — Gaano Ka Ka-Tao?

A simple web-based ESP (Extra Sensory Perception) quiz that checks if *“may bitaw ka sah o wala.”*  
Built using **Golang** for the backend and **HTML/CSS/JavaScript** for a clean, aesthetic frontend.

---

## ✨ Features

- 🎯 10-question empathy & decision-making quiz  
- 👤 Session-based user tracking using cookies  
- 📊 Real-time score calculation  
- 🎨 Minimalist UI (white & gold aesthetic)  
- 🌐 Bilingual questions (English / Filipino)  
- 🏁 Personalized result page based on score

---

## 🛠️ Tech Stack

- **Backend:** Go (Golang)
- **Frontend:** HTML, CSS, JavaScript
- **Templating:** `html/template`
- **State Management:** Cookies + In-memory sessions
